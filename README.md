# CARMIN


![C#](https://img.shields.io/badge/language-C%23-purple)
![.NET](https://img.shields.io/badge/.NET-WinForms-blue)

Sistema de gestión para papelerías y negocios gráficos.

## Tecnologías

- C#
- WinForms
- SQL Server
- Arquitectura en capas

## Funcionalidades

- Gestión de clientes
- Gestión de pedidos
- Gestión de insumos
- Presupuestador
- Reportes
